Версійний контроль

Щоб додати існуючі файли під версійний контроль (на відміну від порожнього каталогу), слід проіндексувати ці файли і зробити перший коміт. Це можна зробити за допомогою декількох команд git add, що визначають файли, які будуть відслідковуватись. Після цього треба виконати команду git commit.

$ git add *.c

$ git add LICENSE

$ git commit -m "Перша версія проекту"
